# 03 - Common FPU Modules

Operações comuns a todas as operações de ponto flutuante.

## Subcategorias:
- **01_unpack**: Desempacotamento de FP (sinal, expoente, mantissa)
- **02_normalize**: Normalização de resultados
- **03_round**: Lógica de arredondamento (RN, RZ, etc.)
- **04_pack**: Empacotamento final em formato FP
- **05_exception_handling**: NaN, Inf, Zero, desnormalizados

## Importância:
Estes módulos garantem conformidade com IEEE 754.
