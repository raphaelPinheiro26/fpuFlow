# 04 - Specific FPU Operations

Operações específicas para processamento de mantissa/expoente.

## Subcategorias:
- **01_mantissa_align**: Alinhamento de mantissas para add/sub
- **02_mantissa_add_sub**: Adição/subtração de mantissas
- **03_exponent_processing**: Cálculo e ajuste de expoentes
- **04_special_cases**: Tratamento de casos especiais

## Contexto:
Estas operações conectam a arithmetic integer com o formato FP.
