# 01 - Integer Arithmetic

Operações aritméticas de inteiros que formam a base da FPU.

## Subcategorias:
- **01_adders**: Somadores (RCA, CLA, etc.)
- **02_multipliers**: Multiplicadores (Array, Wallace, etc.)
- **03_dividers**: Divisores (Restoring, SRT, etc.)
- **04_subtractors**: Subtratores

## Fluxo:
Estes módulos são usados pelas operações de mantissa na FPU.
