# 02 - Primitive Units

Blocos fundamentais usados por múltiplos componentes.

## Subcategorias:
- **01_lzc**: Leading Zero Counters
- **02_barrel_shifters**: Shifters de alta velocidade
- **03_priority_encoders**: Lógica de prioridade
- **04_compressors**: Compressores 3:2, 4:2, etc.
- **05_comparators**: Comparadores numéricos
- **06_muxes**: Multiplexadores

## Uso:
Blocos reutilizáveis para construção de unidades maiores.
